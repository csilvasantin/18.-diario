## Botones AppleScript

Dos botones en la fila superior del perfil "Default Profile":

| Posición | Botón | Script |
|----------|-------|--------|
| `3,0` | ✓ Claude | `approve_claude.applescript` |
| `4,0` | ✓ Codex | `approve_codex.applescript` |

**Comportamiento de cada botón:**
1. Guarda el app en primer plano
2. Enfoca Claude / Codex
3. Envía `Enter` para aprobar el permiso
4. Devuelve el foco al app original

Los `.app` compilados con `osacompile` se ignoran en git (`.gitignore`). Solo se versiona el fuente `.applescript`.

### Bug corregido: error -1719

Al ejecutar desde Stream Deck, `System Events` no encontraba ningún proceso con `frontmost = true` en ese instante exacto, lanzando el error `-1719`. Fix: envolver la consulta en `try/end try`.
