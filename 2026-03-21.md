## Emulador de Spectrum
- Creamos el proyecto `Emulador-de-Spectrum`.
- Objetivo: leer el sonido de una cinta de cassette de ZX Spectrum y convertirlo en datos digitales.
- Se preparó un MVP en Python con:
  - lector de WAV;
  - detección de cruces de cero;
  - estimación de pulsos;
  - conversión aproximada a bits;
  - agrupación en bytes;
  - CLI para probar archivos de audio;
  - tests básicos.
- El proyecto quedó publicado en GitHub como `csilvasantin/Emulador-de-Spectrum`.
- Es una base inicial para evolucionar hacia un decodificador más fiel de bloques de cinta de Spectrum.
