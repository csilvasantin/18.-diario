## Control de Tienda
1. Creamos el proyecto `Control-de-Tienda`.
2. Objetivo: tener una base para controlar productos, stock, alertas y estado operativo de una tienda.
3. Se construyó un MVP con:
   1. servidor Node;
   2. API local;
   3. panel web;
   4. datos de ejemplo;
   5. documentación inicial.
4. La API inicial expone el estado general de la tienda en `/api/store`.
5. También permite actualizar stock por producto.
6. El panel muestra:
   1. resumen;
   2. catálogo;
   3. alertas de bajo stock.
7. Se verificó el funcionamiento local del servidor y de la API.
8. El proyecto quedó publicado en GitHub como `csilvasantin/Control-de-Tienda`.
