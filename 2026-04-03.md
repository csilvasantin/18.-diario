# 3 de abril de 2026

## Proyecto: Memorizer — Agente personal de memoria para el Consejo de Admira Next

Repo: https://github.com/csilvasantin/Memorizer

## Creación del proyecto

- Bot de Telegram (**@Memorizer2Bot**) que actúa como agente personal de memoria para el Consejo de Admira Next.
- Captura mensajes de un grupo de Telegram, los clasifica automáticamente usando Claude API, y los almacena en SQLite con búsqueda full-text (FTS5).

## Stack técnico

- **Python 3.12**, `python-telegram-bot`, Anthropic SDK (Claude Sonnet), SQLite + FTS5, Docker.

## Módulos principales

- `src/bot.py` — Bot principal con comandos `/buscar`, `/resumen`, `/stats`, `/help`
- `src/classifier.py` — Clasificación con Claude API (categoría, resumen, entidades, URLs, fuente)
- `src/storage.py` — SQLite con FTS5 y triggers
- `src/query.py` — Motor de consultas estilo RAG
- `src/council.py` — **NUEVO:** distribución de contenido al Consejo de Admira Next
